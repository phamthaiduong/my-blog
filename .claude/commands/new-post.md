description: Tạo bài viết mới với đúng format frontmatter
---
Tạo file bài viết mới trong data/blog/ với:
- Tên file dạng slug tiếng Anh (kebab-case)
- Frontmatter đầy đủ: title (VI), title_en (EN), date hôm nay, language, tags, summary, draft: true
- Placeholder content cho cả phần VI và EN
- Tạo xong thì mở file để tôi edit
