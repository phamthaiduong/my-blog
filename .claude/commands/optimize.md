description: Kiểm tra và optimize SEO, performance cho blog
---
Kiểm tra theo thứ tự:

1. **SEO**: Mọi bài có summary không? Title < 60 chars không?
2. **Images**: Có alt text không? Đã dùng next/image chưa?
3. **Performance**: Chạy `npm run build` và xem bundle size
4. **Broken links**: Scan tất cả .mdx tìm link hỏng

Báo cáo theo format:
- 🔴 Cần fix ngay
- 🟡 Nên fix
- 🔵 Optional
