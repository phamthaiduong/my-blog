# Component Style Rules

Áp dụng khi tạo React components

## Design constraints
- Chỉ dùng gray/neutral palette — không thêm màu mới
- Không dùng shadow lớn — tối giản
- Spacing: dùng Tailwind spacing scale (4, 6, 8, 12, 16)
- Font: Inter cho UI, Lora cho body text bài viết

## Code conventions
- Functional components + TypeScript
- Props interface đặt trên component
- Không inline style — dùng Tailwind class
